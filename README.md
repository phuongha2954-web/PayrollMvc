# Dự án Quản lý lương (PayrollMvc)
Dự án được xây dựng bằng công nghệ **ASP.NET MVC Core**.
### Các chức năng chính:
* **Quản lý tài khoản:** Đăng nhập, phân quyền người dùng (Role-based).
* **Quản lý nhân viên:** Lưu trữ thông tin và quản lý dữ liệu người dùng.
* **Hệ thống Dashboard:** Thống kê dữ liệu (DashboardRow.cs).
* **Cơ sở dữ liệu:** Sử dụng Entity Framework Core với SQL Server.
